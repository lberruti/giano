Instructions

All mirrors are also browsable via HTTP protocol.

When using Debian configure your /etc/apt/sources.list as it follows:

deb http://giano.com.dist.unige.it/debian/ stable main contrib non-free
deb-src http://giano.com.dist.unige.it/debian/ stable main contrib non-free
